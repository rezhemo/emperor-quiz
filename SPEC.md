# 帝王人格测试 - 规格文档

## 1. 项目概述

- **项目名称**: 帝王人格测试
- **项目类型**: 单页问答应用
- **核心功能**: 通过30道选择题，分析用户性格，匹配最相似的中国知名帝王
- **目标用户**: 对帝王历史感兴趣的用户

## 2. 页面结构

### 页面数量
- 单页应用（无需路由）
- 三个阶段：欢迎页 → 答题页 → 结果页

### 页面流程
```
欢迎页 → 答题页(30题) → 结果页
```

## 3. 功能详情

### 欢迎页
- 标题：测测你和哪位帝王最像
- 简短介绍说明
- 开始测试按钮

### 答题页
- 顶部：进度条（当前题数/总题数）
- 题目显示区域
- 4个选项（单选）
- 点击选项自动进入下一题
- 30道选择题

### 结果页
- 匹配结果：显示最相似的帝王名称
- 匹配度百分比
- 性格特点分析（3-5点）
- 原因分析（为什么得出这个结果）
- 分享按钮（可选）
- 重新测试按钮

## 4. 帝王类型定义

根据答题结果，匹配以下帝王类型：

1. **秦始皇** - 开创型/统治型
2. **刘邦** - 务实型/平民皇帝
3. **李世民** - 智慧型/文治武功
4. **朱元璋** - 草根逆袭型
5. **康熙** - 学习型/全才型
6. **武则天** - 打破常规型
7. **乾隆** - 艺术型/享乐型
8. **曹操** - 枭雄型/复杂人格

## 5. 计分系统

每道题有4个选项，分别对应不同帝王倾向：
- A → 帝王1
- B → 帝王2
- C → 帝王3
- D → 帝王4

30题结束后，统计每个帝王的得分，最高分即为匹配结果。

## 6. 题目设计方向

涵盖维度：
- 领导风格
- 处理冲突方式
- 对财富态度
- 对权力态度
- 个人爱好倾向
- 处事哲学
- 情感处理
- 风险偏好

## 7. 视觉设计

- 配色：金色+深红/暗色背景（中国古典风格）
- 字体：衬线体（思源宋体或系统默认）
- 风格：典雅、大气、古典
- 动效：淡入淡出过渡

## 8. 技术实现

- 纯 HTML + CSS + JavaScript（单文件）
- 无需后端
- 无需数据库
- 可部署到任意静态托管服务
